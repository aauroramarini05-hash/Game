# xDustAtom City 3D Demo

Demo 3D in Three.js con:

- Intro "xDustAtom Present".
- Città procedurale con edifici, strade, alberi e traffico animato.
- UI avanzata con lingua IT/EN, qualità grafica, ciclo giorno/notte e traffico.
- Limite FPS regolabile in tempo reale.

## Avvio

Essendo una demo statica, basta aprire la cartella con un web server locale:

```bash
python3 -m http.server 8080
```

Poi aprire `http://localhost:8080`.

## Controlli

- **Mouse**: orbita camera
- **W/A/S/D**: movimento camera
- **Pannello UI**: impostazioni avanzate
